in ASP.Net Projects you must disable shadow copy
To disable shadow copy add following setting in web.config file:

<configuration>
  ...
  <system.web>
    ...
    <hostingEnvironment shadowCopyBinAssemblies="false" />
    ...
  </system.web>
  ...
</configuration>
